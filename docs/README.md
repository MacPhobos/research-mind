# Research-Mind Monorepo

Full-stack application: FastAPI backend + SvelteKit frontend.

## Structure
- `research-mind-service/` - Python FastAPI backend
- `research-mind-ui/` - TypeScript SvelteKit frontend
- `docs/` - Shared documentation

## Quick Start
```bash
make install
make dev
```

Service: http://localhost:15010
UI: http://localhost:15000

See `CLAUDE.md` for detailed development guide.
